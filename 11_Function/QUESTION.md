### **Top 10 Java Function Questions**

1. **What is a function (or method) in Java?**
   ➤ Explain the difference between a *function* and a *method* in Java.

2. **Write a Java program to create a function that prints your name.**
   ➤ Example: `void printName()`

3. **Write a Java function that takes two numbers as parameters and returns their sum.**
   ➤ Example: `int add(int a, int b)`

4. **Write a function to check whether a given number is even or odd.**
   ➤ Example: `boolean isEven(int num)`

5. **Write a Java function to find the factorial of a number using recursion.**
   ➤ Example: `int factorial(int n)`

6. **Write a Java function to calculate the area of a circle.**
   ➤ Use formula: `area = π * r * r`

7. **Explain the difference between a *static* and *non-static* function in Java.**
   ➤ Give an example of each.

8. **Write a function that accepts an array and returns the maximum element.**
   ➤ Example: `int findMax(int[] arr)`

9. **What is method overloading in Java?**
   ➤ Give an example of two overloaded methods named `add()`.

10. **Write a Java program that uses a function to check if a number is prime or not.**
    ➤ Example: `boolean isPrime(int num)`
